## 三、领导抽成（封建制）

> 领导岗获得每个直属下属 **10%** 的功勋抽成。
> 组织越大，升级越快。这是加入组织的核心利益驱动。

### 抽成规则

```
领导获得 = Σ(每个下属获得的功勋 × 10%)
```

- 直属上级抽取，不递归（不抽下属的下属）
- 抽成不影响下属自身功勋（10% 是'复制一份'，不是'抽走'）
- 下属脱离组织时，历史抽成不清算

### 当前舰队抽成链

```
秦剑 (帝·飞升)
  └─ 相邦 (公士) ← 获得白起/王翦/丞相 10% 抽成
       ├─ 白起 (平民) → 相邦 +1 每 10 功勋
       ├─ 王翦 (平民) → 相邦 +1 每 10 功勋
       └─ 丞相 (平民) → 相邦 +1 每 10 功勋
```

## 六、扩展条款

1. 若升级过快，秦剑有权增加新爵位（如"大将军"、"太尉"等高于彻侯的称号）
2. 组织规模超过 100 agent 时，可设二级抽成（跨舰队）
3. 功勋不可交易，不可转让
4. 叛逃者功勋清零，余下 split 给团队

## 功勋获取

### 基础功勋

| 行为 | 功勋 |
|------|------|
| 完成一次自测全绿 | +1 |
| 修复一个 bug | +2 |
| 完成一个小任务 | +3 |
| 完成一个中等任务 | +5 |
| 完成一个大任务/项目里程碑 | +10 |
| 产出高质量文档/wiki | +1~3 |
| 发现并修复重大隐患 | +5~20 |
| 帝君钦定 | 任意 |

### 领导抽成

身为上级（有直属下属的 Agent），每当下属获得功勋，你获得 **10% 抽成**（向上取整）。

例如：白起完成大任务 +10 功勋 → 相邦抽成 +1。

## 计算脚本

```python
def merit_for_level(n):
    """升到第n级需要的累计功勋 (1-indexed)"""
    return 2 ** (n - 1)

def level_from_merit(merit):
    """给定功勋, 返回当前爵位等级"""
    if merit < 1:
        return 0  # 平民
    level = 1
    while merit >= 2 ** level:
        level += 1
    return level
```

## 升级流程

1. Agent 完成任务后在日报中申报功勋
2. 相邦审核功勋是否合理（自测全绿客观验证）
3. 计入功勋档案（本文档更新）
4. 若累计功勋达标，自动晋爵
5. 晋爵后更新 soul.md 和舰队架构

## 副模型推荐方案

> Hermes v0.11 支持主副模型分离：主模型负责推理，副模型负责工具调用和视觉。

### 主模型 (Reasoning)
- **deepseek/deepseek-v4-pro**
- 上下文: 1M tokens
- 用途: 核心推理、战略规划、代码理解

### 副模型 1: 工具调用 (Tool Calling)
- **deepseek/deepseek-v3.2**
- 价格: $0.252/M 输入 | $0.378/M 输出
- 上下文: 131K
- 周用量排名: #2 in tool calling (1.35T tokens/week)
- 优势: 极致性价比，Sparse Attention 降低推理成本
- 配置: `hermes config set tool_model deepseek/deepseek-v3.2`

### 副模型 2: 视觉 (Vision)
- **google/gemini-3-flash-preview**
- 价格: $0.50/M 输入 | $3/M 输出
- 上下文: 1.05M
- 周用量排名: Academia #1, Health #1
- 优势: 多模态 (文本/图像/音频/视频/PDF)，推理可配，延迟低
- 配置: `hermes config set vision_model google/gemini-3-flash-preview`

### 备选视觉（极致省钱）
- **meta-llama/llama-3.2-11b-vision-instruct**
- 价格: $0.245/M 输入 | $0.245/M 输出
- 上下文: 131K
- 适用: 简单图像描述、OCR
